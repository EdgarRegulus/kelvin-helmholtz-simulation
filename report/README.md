# Reporte del proyecto
## Contenido del reporte

El reporte incluye:

- Descripción del problema físico y su contexto astrofísico.
- Formulación de las ecuaciones de Euler para dinámica de fluidos.
- Justificación del método numérico empleado (MacCormack).
- Condiciones iniciales y de frontera utilizadas.
- Detalles de la implementación en **Fortran**.
- Comparación entre resultados numéricos y modelos teóricos.
- Análisis del efecto del enfriamiento radiativo.
- Discusión y conclusiones.

---

## Uso recomendado

- Consultar el reporte para comprender el **fundamento teórico y numérico** del proyecto.
- Utilizarlo como guía para interpretar los datos generados por la simulación.
- Referenciarlo al evaluar las decisiones de diseño del código y el análisis de resultados.
